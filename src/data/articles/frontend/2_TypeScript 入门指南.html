TypeScript 入门指南

TypeScript 是 JavaScript 的超集，它添加了静态类型检查和其他高级特性，使代码更加健壮和可维护。

类型系统

TypeScript 的核心特性是其类型系统，它允许我们为变量、函数参数和返回值指定类型。

// 基本类型
let isDone: boolean = false;
let age: number = 18;
let name: string = "张三";

// 接口
interface Person {
  name: string;
  age: number;
}

// 函数
function greet(person: Person): string {
  return `Hello, ${person.name}!`;
}

泛型

泛型允许我们编写可重用的代码，适用于多种类型。
